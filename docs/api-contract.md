# API Contract (SUT)

Base URL (local): `http://localhost:8080`

## Data model: TestRecord

### Fields
- `id` (string, UUID) – generated by server
- `name` (string, required, 3..100)
- `status` (string, required) – one of: `DRAFT`, `IN_PROGRESS`, `PASSED`, `FAILED`
- `createdAt` (string, ISO-8601) – generated by server
- `updatedAt` (string, ISO-8601) – generated by server

### Create request body
```json
{
  "name": "Leakage current test - Luminaire A",
  "status": "DRAFT"
}
```

## Endpoints

### 1) Health check
- Method: `GET`
- Path: `/health`
- Description: Confirms that the API is running and reachable.

**Success response**
- HTTP Status: `200 OK`
```json
{ "status": "UP" }
```

### 2) Create TestRecord
- Method: `POST`
- Path: `/test-records`
- Description: Creates a new test record.

**Request body**
```json
{
  "name": "Leakage current test - Luminaire A",
  "status": "DRAFT"
}
```
